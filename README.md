# V2Ray Subscription Aggregator

> ⚠️ **This project is no longer maintained.** The upstream subscription sources used by this tool have all gone offline or changed their APIs. The code is kept here for archival purposes only.

~~A simple script that aggregates free V2Ray subscription links from public sources.~~

## Status: DEPRECATED (2025-03)

This project relied on several public free node sharing websites that no longer exist:
- `nodefree.org` — domain expired
- `bulink.xyz` — shut down
- `freefq.com` — blocked

Since all upstream sources are dead, this tool no longer produces any useful output. If you're looking for alternatives, check the V2Ray community forums.

## Historical Usage (no longer works)

```bash
pip install -r requirements.txt
python3 v2ray_aggregator.py --top 10
```

The script would fetch subscription links, decode base64 node lists, and output a merged file. None of this works anymore since the sources are offline.
